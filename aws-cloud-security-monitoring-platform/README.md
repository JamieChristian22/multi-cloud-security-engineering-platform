# AWS Cloud Security Monitoring & Detection Platform

Designed and implemented a cloud security monitoring solution using AWS CloudTrail, CloudWatch, SNS, IAM, and AWS Config.

## Objectives
- Monitor privileged account activity
- Detect unauthorized access attempts
- Track IAM changes
- Monitor security group modifications
- Generate real-time alerts
- Maintain compliance visibility
