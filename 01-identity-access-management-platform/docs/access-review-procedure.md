# Access Review Procedure

## Review Frequency

Access is reviewed quarterly for all privileged users and monthly for break-glass accounts.

## Review Steps

1. Export users, groups, roles, and assignments from AWS, Azure, and GCP.
2. Identify administrative permissions.
3. Verify business justification for each privileged assignment.
4. Remove stale users and unused service accounts.
5. Rotate keys for approved service accounts.
6. Save evidence in the evidence folder.

## Review Outcomes

- Approved access
- Access removed
- Access reduced
- Exception created with expiration date
