# AWS Monitoring Compliance Matrix

| Control | Service | Evidence |
|---|---|---|
| Administrative actions logged | CloudTrail | Event history / trail |
| Security findings centralized | Security Hub | Summary and findings |
| Threat detection enabled | GuardDuty | Detector and findings |
| Alerts configured | CloudWatch | Alarm list |
| Notifications configured | SNS | Topic and subscriptions |
| Configuration changes monitored | EventBridge | Rules |
