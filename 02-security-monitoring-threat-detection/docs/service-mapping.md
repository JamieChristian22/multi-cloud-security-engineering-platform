# Multi-Cloud Service Mapping

| Monitoring Area | AWS | Azure | Google Cloud |
|---|---|---|---|
| Metrics | CloudWatch Metrics | Azure Monitor Metrics | Cloud Monitoring |
| Logs | CloudWatch Logs | Azure Monitor Logs | Cloud Logging |
| Audit Logs | CloudTrail | Activity Log | Cloud Audit Logs |
| Alerts | CloudWatch Alarms | Azure Monitor Alerts | Alerting Policies |
| Notifications | SNS | Action Groups | Notification Channels |
| Threat Detection | GuardDuty | Defender for Cloud | Security Command Center |
| Security Posture | Security Hub | Defender for Cloud | Security Command Center |
| Automation | EventBridge / Lambda | Event Grid / Logic Apps | Eventarc / Cloud Functions |
