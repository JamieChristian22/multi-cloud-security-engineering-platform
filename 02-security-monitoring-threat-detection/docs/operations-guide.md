# Operations Guide

## Daily Checks

- Review cloud security findings.
- Review alert status.
- Confirm log ingestion is active.
- Check failed authentication and privileged activity.
- Review configuration changes.

## Weekly Checks

- Review IAM changes.
- Review security posture recommendations.
- Validate alert routes.
- Review noisy or duplicate alerts.
- Update runbooks based on real events.

## Monthly Checks

- Review monitoring coverage.
- Review compliance dashboards.
- Validate retention settings.
- Test incident notification flow.
