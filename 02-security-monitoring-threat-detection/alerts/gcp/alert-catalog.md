# Google Cloud Alert Catalog

| Alert | Signal | Severity | Action |
|---|---|---|---|
| High CPU | Cloud Monitoring metric | Warning | Notify operations |
| IAM Policy Changed | Cloud Audit Logs | High | Review actor and role |
| Service Account Key Created | Audit log event | High | Validate or revoke |
| Public Bucket Change | Storage IAM change | Critical | Remove public access |
| SCC High Finding | Security Command Center | High | Start investigation |
| Logging Sink Deleted | Audit log event | High | Restore log routing |
