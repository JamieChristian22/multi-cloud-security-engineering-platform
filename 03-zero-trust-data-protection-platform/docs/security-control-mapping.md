# Security Control Mapping

| Control | AWS | Azure | Google Cloud |
|---|---|---|---|
| Least Privilege | IAM Policies | Azure RBAC | Cloud IAM |
| Encryption at Rest | AWS KMS | Storage Encryption / Key Vault | Cloud KMS |
| Secret Storage | Secrets Manager | Key Vault | Secret Manager |
| Public Access Prevention | S3 Block Public Access | Disable Blob Public Access | Public Access Prevention |
| Audit Logging | CloudTrail | Activity Log | Cloud Audit Logs |
| Compliance Monitoring | Security Hub / Config | Defender for Cloud / Policy | Security Command Center |
| Data Classification | Macie | Purview / Defender | Sensitive Data Protection |
