# Operations Guide

## Daily Operations

- Review data access logs.
- Check for public storage exposure.
- Review failed access attempts.
- Monitor secret access events.
- Validate key rotation alerts.

## Weekly Operations

- Review privileged role assignments.
- Review storage access policies.
- Validate encryption settings.
- Review cloud security posture findings.

## Monthly Operations

- Review all service accounts and managed identities.
- Rotate secrets where required.
- Review unused access.
- Update policies and runbooks.
