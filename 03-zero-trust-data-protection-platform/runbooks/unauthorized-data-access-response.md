# Runbook: Unauthorized Data Access Response

## Trigger

Unexpected access to sensitive cloud data.

## Response Steps

1. Identify principal, source IP, resource, and action.
2. Determine whether access was approved.
3. Revoke unnecessary permissions.
4. Review identity and access history.
5. Check for data exfiltration indicators.
6. Escalate if customer or regulated data was accessed.
7. Record final incident summary.

## Closure Criteria

Access is contained, permissions are corrected, and investigation is documented.
